🧠 Neural Network Model

  • Algorithm: Multi-Layer Perceptron Classifier (MLPClassifier).
  • Architecture: Two dense hidden layers containing 64 and 32 neurons respectively to decode non-linear patterns.
  • Data Pipeline: Features (Volatility, Trend, Volume) are rigorously scaled using StandardScaler after an 80/20 stratified split to prevent data leakage.
  • Training Optimization: Model solves iterations up to 1000 epochs, incorporating early stopping mechanisms to resist overfitting, and is validated using 5-Fold Cross Validation.

📚 Fuzzy Membership Engine

Mamdani Inference translates the continuous NN score (0-10) into soft structural bounds.

  • Low Risk: Trapezoidal boundary [0, 0, 3, 5]
  • Medium Risk: Triangular boundary [3, 5, 7]
  • High Risk: Trapezoidal boundary [5, 7, 10, 10]
